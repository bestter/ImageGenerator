💡 **What:** Replaced the LINQ `Select` chain and `.ToArray()` call in `Form1.PrepareReferenceImagesAsync` with a standard `foreach` loop iterating over a pre-allocated `List<Task<ImageUrlObject?>>`.
🎯 **Why:** To prevent unnecessary memory allocations. Using LINQ `.Select().ToArray()` to map items to asynchronous tasks creates intermediate enumerators, arrays, and incurs closure overhead per item. Moving to a pre-allocated list and a standard loop reduces Garbage Collection (GC) pressure on the UI thread when generating images with multiple references.
📊 **Impact:** Eliminates intermediate array and enumerator allocations during the critical path of setting up asynchronous image processing tasks.
🔬 **Measurement:** Reduced GC pauses during the "Generate" click event when multiple image files are attached.
